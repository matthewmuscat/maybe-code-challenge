# Code challenge: Frontend

[View 1 hour demonstration video here](https://www.loom.com/share/39422ae0b79746e88c2a4e67db795c3a)

## Stack

- Next.JS
- TypeScript
- TailwindCSS

## Getting started

```
yarn install
```

## Running locally

```
yarn dev
```

Once started, the app should be available via [localhost:3000](http://localhost:3000)
